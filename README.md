#vue-more-pages
基于vue-cli的多页面应用


默认进入
src->page->index->index.html

测试页面地址：
src->page->test->test.html

运行地址:localhost:8080/test.html


``` bash
项目地址：（git clone）

git clone https://github.com/hujinbin/Vue-sell.git
 
 
# 通过npm安装本地服务第三方依赖模块(需要已安装Node.js)
npm install

# 本地运行于 localhost:8080
npm run dev

# 打包文件
npm run build

```
